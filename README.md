# Prime factorization in Java

A simple program that gets the prime factors of a given number.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Enter `javac` into the terminal to see if you need to install JDK (if you get an error you need to install it)
2. Download the latest version of JDK from [adoptium.net](https://adoptium.net/)
3. Enter `javac primFact.java` in the terminal to compile primFact.java

Instructions for using the software:

## To run:
1. Open a terminal
2. In the terminal type and enter `java primFact`
3. Follow prompts in the terminal

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:
